# 인공지능 수학 HW1

인공지능 수학 수업 시간에 아무 영상에 SVD 알고리즘을 적용하는 과제가 있었다. 때마침 DiT에 흥미를 갖고 찾아보고 있었던 때라 DiT의 feature에 SVD를 적용해보면서 그 의미를 생각해보았다.

## 1. Full SVD, thin SVD, Compact SVD, Truncated SVD 구현하기

Python과 pytorch를 활용하여 4개의 서로 다른 특잇값 분해 함수를 제작하였다. 해석의 용이성을 위해 분해 대상 행렬의 크기를 $$m$$, $$n$$으로 정의하고, 모든 행렬을 $$U \Sigma V^T$$ 꼴로 분해하였다.

다음은 truncated_SVD 코드이다.

```python
def truncated_SVD(A, rank):
  A = A.to(device, dtype=torch.float32)
  m, n = A.shape

  eigen_vals, V = torch.linalg.eig(A.T @ A) # V.shape = m,m
  eigen_vals = torch.clamp(eigen_vals.real, min=0) # 복소수 및 노이즈 제거를 위한 clamp
  eigen_vals, idx = torch.sort(eigen_vals, descending=True) #고윳값 내림 차순 정렬

  valid_idx = eigen_vals > 1e-8 #threshold 적용을 통한 계산 오차 방지

  if rank <= min(m, n):
    singular_vals = torch.sqrt(eigen_vals[valid_idx][:rank])
  else:
    print("ERROR: rank should be lower than rank(A)")
```

 이때 pytorch로 singular value를 구하는 과정에서 수치적인 오류가 발생하였다. 수치상 발생한 노이즈를 제거하기 위해 0에 아주 가까운 값(1e-10 이하)을 무시하는 threshold를 설정했다. 그러나 0에 가까운 eigen value 노이즈에 루트를 취하는 과정에서 threshold를 넘어버려 Compact SVD나 Truncated SVD를 구할 때 정확한 랭크 값을 구할 수 없었다. 따라서 eigen value 자체에 threshold를 적용함으로써 수치 상 오류를 방지할 수 있었다.

```python
V = V.real[:, idx] #고유 벡터 중 허수 노이즈 제거 및 내림차순으로 정렬
Vt = V[:, :r].T

r = rank

U = torch.zeros((m, r), device=device) #Truncated을 위한 U 선언
for i in range(r):
    if singular_vals[i] > 1e-8:
        U[:, i] = (A @ V[:, i]) / singular_vals[i] #고유 벡터 행렬 U 구하기

Sigma = torch.zeros((r, r), device=device) #rank에 맞는 sigma 할당
Sigma = torch.diag(singular_vals[:r])
```

위와 같이 Truncated SVD를 포함한 4개의 SVD 함수에 대해 임의의 pytorch tensor 행렬, 임의 이미지에 실험을 진행하여 함수가 잘 동작하는지 검토하였다.

## 2. 입력 데이터 특성에 따른 $$\Sigma$$ 분석하기

 DiT 모델은 최근에도 사용되고 있는 최신 모델 중 하나이며 높은 품질의 이미지를 생성하는 t2i 이미지 생성모델로써 아직까지도 활발한 연구가 이루어지고 있다. 그러나 여전히 DiT의 각 time step과 block의 역할과 의미에 대해선 블랙 박스로 취급되고 있으며 밝혀지지 않은 바가 많다. 한편, 특잇값 분해는 선형 변환의 방향과 크기를 분해해주는 기술로, 모델의 가중치 행렬이 입력 출력 간 어떤 방향 성분을 얼마나 강조하는지 알 수 있다. 최근에는 특이값 분해를 이용해 경제적인 fine-tuning 방식이 제시되거나, 모델을 효과적으로 압축하는 연구가 이루어지고 있다. 

 본 실험에서 DiT 모델의 step과 block별 feature 값들에 SVD를 활용하여 DiT 모델을 부분적으로 분석하였다. 실험에서는 Stable Diffusion v.3을 사용하였으며, 실험 환경을 고려하여 inference step은 20으로 설정하였다. 또한 전체 step 중 1, 10, 19 step에 대해서만, 전체 23개의 block 중 1, 6, 12, 18, 22 번째 블록에 대해서만 우선적으로 분석하였다. 마지막으로 각 블록의 feature는 $$(\mathrm{Width}*\mathrm{Height}, \mathrm{Channel})$$형태의 행렬로 취급하였다. 결과에 따르면 대부분의 step과 block에서 특잇값 $$\sigma_{i < 400}$$이 유의미한 값을 담고 있었으며, 프롬프트와 관계 없이 첫 번째 특잇값이 매우 큰 형태를 보였다. 이는 Sd v.3 모델의 $$\sigma_0$$에 대응하는 특이 벡터 $$v_0$$이 모델의 baseline과 연관이 있음을 시사한다. 또한 후반부 블록일수록 step이 진행됨에 따라 유의미한 특잇값의 수가 증가하는 경향을 보였다.

  더불어 각 step과 block에서 $$\Sigma$$의 유효 dimensionality를 알아보기 위해 participation Ratio(Spectral Effective rank)를 활용하였다. 다양한 길이의 프롬프트에 실험해본 결과 프롬프트 길이와 무관하게 대부분의 이미지들의 평균 Spectral effective rank가 100 이하였다.

$$
\mathrm{Participation \ Ratio} = \frac{(\sum_i \sigma_i)^2}{\sum_i \sigma^2_i}

## 3. Truncated SVD를 2번과 연계하여 그 성능을 분석하기

Truncated SVD를 이용하여 각 step과 block에서의 latent feature를 압축할 수 있다. 이때 유의미한 압축을 위해 Frobenius norm 기반으로 상대적 손실률을 계산했다. $$F_t^{(b)} \in \mathbb{R}^{H \times W}$$를 시간 step $$t$$에서 블록 $$b$$의 원본 feature 행렬로, 같은 공간에서의 truncated SVD로 복원한 feature는 $$\hat{F}_t^{(b)}$$로 나타낼 때, 다음과 같이 Frobenius norm을 이용해 상대 압축 손실률 $$\hat{L}$$을 계산할 수 있다.

$$
\hat{L}^{(b)}_t = \frac{||F_t^{(b)} - \hat{F}_t^{(b)}||_F }{||F^{(b)_t}||}, \ \ ||F^{(b)}_t||_F = \sqrt{\sum^H_{i=1}\sum^W_{j=1}(F_T^{(b)})^2_{ij}}
$$

그 결과 $$\mathrm{rank}(\Sigma)\ge 400$$일 때 평균 상대 압축 손실률이 5%이하로 떨어졌다. **표3**에서 볼 수 있듯 낮은 step에서 feature는 높은 redundancy를 보인 반면, 높은 step에서는 낮은 redundancy를 보였다. 특히 block-22에서 평균적으로 높은 손실률을 보였는데, 후반부 block이 보통 이미지의 디테일에 해당하는 고주파 정보를 다룬다는 점을 고려할 때 흥미로운 결과이다. 또한 **그림2**에서 알 수 있듯 이러한 차이는 시각적으로도 나타나며, block과 step에 따라 동일한 압축에 대해서 다른 시각적 차이를 보인다. **그림2** 왼쪽 4열 이미지들을 확인하면 높은 압축에서 고양이 형상이 뚜렷해지는 점은 인상적이다.

## 4. 결론

 이번 실험을 통해 Dit 모델의 이미지 생성 과정에서 비등방적인 압축 가능성이 있음을 수치적으로 확인했다. 이는 DiT 모델의 부분적인 경량화 가능성을 시사하며, 모델의 finetuning 과정에서도 보다 보다 효율적인 학습이 가능함을 보여준다.

 추가적으로, 본 실험을 모든 step과 block에 대해 확장 적용해봄으로써 앞서 발견된 압축적 비등방성을 보다 구체적으로 검토할 수 있다. 더불어 단일 채널 값에 대해 SVD를 적용해봄으로써 각 채널의 의미하는 바를 간접적으로 확인할 수 있다. 또한, 이미지 feature에 대해서만 실행한 결과임으로 text feature에 확장 적용이 가능하다. 마지막으로, 실험에서 사용한 feature를 $$(C, H*W)$$로 정의하고 SVD를 적용하여 공간적 차원의 압축까지 고려할 수 있다.
